CEDES-16

I wanted to build a simple fantasy console based on a somewhat cursed 16-bit ISA.
The instruction set is all over the place, and game cartridges are 64KB RAM snapshots.
The emulator is written in C, and the games are written in a custom assembly language.
You can read ramble.txt for some of my thoughts on the design and implementation of the emulator and assembler.
To be honest, nothing is very structured right now, but I am deep into trying to make it work first.
